WI Recycle

An Android app that helps people learn about recycling.

WI Recycle was created in the Fall of 2015 in collaboration with the First Lego League team Code Orange, FLL Team #16152. Code Orange is a group of elementary and middle school students who came up with an idea about an app to help educate people about recycling in Wisconsin. The collaboration was done with the students from Code Orange to write an Android app from scratch to show what recycling resources are available in each county of Wisconsin.

You can download WI Recycle here.
